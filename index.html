Untuk mengenang masa-masa tempel dan edit muka fl facebook, maka saya sebagai akun Ahmad Marcel Albert membuat game ini dengan style VHS Player.

PERINGATAN:

Jangan mencoba mengakses dokumen-dokumen game tanpa izin level 4. Pelanggaran akan dilacak dan ditindak sesuai prosedur pengamanan. Mengakses game ini berarti anda setuju dengan efek samping yang diterima yaitu mual-mual, pusing kepala, kecanduan, halusinasi dan depresi berat. Sekalinya anda mengakses game ini maka tidak ada jalan kembali kesini.
